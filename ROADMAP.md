Grow from 22★ to 1000★ through better packaging, OSS trust signals, and systematic distribution.

### 16–25 May — Product proof points
- Prepare v1.0.0 tag + release notes

### 26 May–08 Jun — First distribution wave
- Multi-channel launch posts (HN, Reddit, Dev.to/Hashnode, X/LinkedIn)

### 09–22 Jun — Viral loop and community
- Run 2 public live demo/AMA sessions

- Run mini challenge around review prompts / improvement cases

### 23 Jun–01 Jul — Final sprint
- Daily "Road to 1k" updates
- Publish social-proof recap post
- Coordinate partner reposts from niche communities

- README CTR
- Stars/day
- Stars per 1,000 views
- Source of stars (HN/Reddit/X/Dev.to/YouTube)
- External contributors, PRs, and Discussions activity
