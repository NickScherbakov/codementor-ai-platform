# Support

## How to get help

- **Bug reports:** open a GitHub Issue using the bug template.
- **Feature requests:** open a GitHub Issue using the feature template.
- **Q&A / ideas:** use GitHub Discussions (if enabled in repository settings).

## Security issues

Please do not post security vulnerabilities publicly. Follow [SECURITY.md](SECURITY.md).

## Response targets

- Initial triage for issues: within 3 business days
- Follow-up for reproducible bugs: within 7 business days
